# CHECK24 App
iOS Application based on MVVM and Combine.

# About
This project is for CHECK24 assignment.

# Requirements
* Xcode 11 or later with Swift 5.0
* iOS 13 or later

# Getting Started
* Clone or download
* Pod install
* Build and Run

# Technical Features
* MVVM Arhitecture
* Coordinator Pattern
* Using property wrapper for userdefault
* Combine
